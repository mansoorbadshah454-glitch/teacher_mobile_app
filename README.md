# Teacher App (Mobile)

A Flutter application for teachers to manage students, attendance, and more.

## Architecture
- **State Management**: Riverpod (Providers)
- **Navigation**: GoRouter
- **Backend**: Firebase (Auth, Firestore)
- **Design**: Material 3 with Google Fonts (Inter)

## Setup
1. Ensure `google-services.json` is placed in `android/app/`.
2. Run `flutter pub get`.
3. Run the app on an Emulator or Device.

## Features
- Login / Authentication
- Dashboard
- Attendance Marking
- Student List
